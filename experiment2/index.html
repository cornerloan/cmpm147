Experiment 2 - Living Impressions

Description

This experiment creates generated art based off of the image.

The image has a few key features: trees, a mountain, a pink and blue sky, and a river.

I added a cloud and sun to add life to the image.

Technical

I had to split this up into multiple sections.

The Trees surround the river, so I created a river and sprinkled trees on both sides of the river.

The river itself is a cascade of rounded edge rectangles, which move around left and right to show a winding river.

The mountain is a single line with bumps added to simulate ridges.

The sky is a pink box, with random blue opacity ellipses added to simulate a sunset.

The clouds move based on time, and loop through the sky.

The sun moves in the sky based on a sin wave, and moves X position based on the X position of the mouse.

Reflection

This project started out easy, thanks to the code provided. The mountain and trees use the same code as the provided example.

I added a river, and made it wind which actually didn't take too long. Afterwards, I had to make sure trees weren't inside the river. I changed this by tweaking the tree placement to not include the middle third of the canvas.

Next, I added the cloud which moves to the right based on time, and loops back to the left side once the right side of the canvas is reached.

The sun took a while, but overall I learned a lot about manipulating sin waves based on how I want objects to move in an arch.

Overall, I'm not too happy with the opacity of the blue and pink sky. I wish I knew a better way to implement the opacity, but the level of quality matches the rest of the basic shapes of the project
