Experiment 3 - Alternate Worlds



Description

This assignment was done to procedurally generate maps.
One was a dungeon crawler type world, with two connected rooms.
The other is an overworld showing a river and trees.

Technical

This program works by generating a two dimensional array of spaces, and procedurally assigning values that represent different qualities for the maps. For instance, in the overworld map a "T" is a tree, a "R" is water, and a "_" is grass.
From there, a graphical map is created that uses autotiling to convert the array of symbols into artwork generated from an image.

Reflection

The biggest difficulty with this project was learning how autotiling works, and attempting to map out an algorithm for correctly identifying neighboring spaces.
The walls of the dungeon look at surrounding tiles to determine which way to face a wall, or to use a corner wall. Once I got this technique down though the project wasn't bad.
